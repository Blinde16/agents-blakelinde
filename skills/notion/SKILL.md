# Notion Skill

Interact with the Notion Content Calendar database to read, plan, and insert new editorial items.

## Usage

You have access to the python implementation in `scripts/notion_calendar.py`. 

**Functions Available:**
- `execute_notion_create_calendar_entry`
- `execute_notion_update_calendar_entry`

**APPROVAL REQUIRED:**
Creating or updating items in Notion requires explicit user approval ([Y/N]) prior to execution.
