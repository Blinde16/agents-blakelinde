# Social Media Skill

This skill allows you to draft and manage social media posts.

## Usage

You have access to the python implementation in `scripts/social.py`.

**Functions Available:**
- `execute_publish_social_post`

**APPROVAL REQUIRED:**
You MUST check with the user before publishing or scheduling any social post. Present the exact copy and platform, ask for [Y/N], and only execute upon approval.
